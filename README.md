# Future-Proof Portfolio Project

A modern, responsive portfolio website built with React and Vite, showcasing Phase 1 of the Future-Proof Portfolio Project for an Information Technology student.

## Features

- Modern React architecture with functional components
- Smooth scrolling navigation
- Responsive design
- Gradient backgrounds and modern UI
- Component-based structure for easy maintenance

## Tech Stack

- React 18
- Vite (for fast development and optimized builds)
- Modern CSS with CSS Variables

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the local URL shown in the terminal (usually `http://localhost:5173`)

## Build for Production

To create an optimized production build:

```bash
npm run build
```

The built files will be in the `dist` folder, ready to be deployed.

## Deployment

This project can be easily deployed to Vercel, Netlify, or any other static hosting service.

For Vercel:
1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically detect Vite and deploy your site
